# Series Data

Эта папка содержит JSON файлы с данными для каждой серии автомобилей.

## Структура файлов

Каждый файл должен быть назван по формату slug серии (например, `m8-f91-m8-convertible.json`) и содержать следующие поля:

```json
{
  "title": "Название серии",
  "subtitle": "Подзаголовок",
  "description": "Полное описание серии",
  "advantages": [
    "Преимущество 1",
    "Преимущество 2",
    "Преимущество 3"
  ],
  "problems": [
    "Проблема 1",
    "Проблема 2",
    "Проблема 3"
  ],
  "image": ""
}
```

## Примечания

- **title**: Полное название серии (например, "BMW M8 F91 Convertible")
- **subtitle**: Краткое описание одной строкой
- **description**: Полное описание серии (2-3 предложения)
- **advantages**: Массив преимуществ (обычно 4-6 пунктов)
- **problems**: Массив частых проблем (обычно 4-6 пунктов)
- **image**: Оставляйте пустым - изображение определяется автоматически на основе модели

## Как добавить новую серию

1. Создайте новый JSON файл с именем, соответствующим slug серии
2. Скопируйте структуру из `_template.json`
3. Заполните все поля данными
4. Сохраните файл

## Пример slug

Для URL `/cars/bmw-x3/x3-lci-f25-b47-18d20d` создайте файл `x3-lci-f25-b47-18d20d.json`
